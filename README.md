#CocoaAsyncSocket-Server

同一WiFiネットワーク上なら相手端末のIPを入力するだけでサーバーを介さずにiOS間でリアルタイム通信ができるサンプルアプリケーションです。

##How to use

1. ２台のiOS端末を準備し、Server側とClient側でそれぞれアプリケーションをインストール

2. Client側から相手端末のIPとメッセージを入力

3. Server側でラベルに送信したメッセージが反映される
